Text Reader Privacy Policy

Effective date: March 30, 2026

Text Reader is a Chrome extension that allows users to highlight text on a webpage and hear it read aloud using the browser's built-in speech synthesis.

Information the extension accesses

Text Reader may access:

  • Text that the user explicitly highlights on a webpage
  • The active tab context needed to run the extension on the current page
  • Temporary playback state, such as whether reading is active or paused

How information is used

This information is used only to provide the extension's core functionality:

  • Reading highlighted text aloud
  • Supporting pause, play, and stop controls
  • Showing playback status in the extension popup

Data storage

Text Reader may store limited temporary playback state locally in Chrome storage. Text Reader does not store highlighted text on remote servers.

Data sharing

Text Reader does not:

  • sell user data
  • share user data with third parties
  • use user data for advertising
  • use user data for analytics
  • use user data for creditworthiness or lending decisions

Remote processing

Text Reader processes highlighted text locally in the browser using built-in browser speech synthesis. The extension does not transmit highlighted text to external servers.

Security

Text Reader is designed to minimize data access and uses only the permissions necessary to provide its functionality.

# Chrome Web Store Listing Copy

## Extension name
Text Reader

## Short description
Highlight text on any webpage and listen with simple read, pause, play, and stop controls.

## Detailed description
Text Reader lets you highlight text on any webpage and hear it read aloud instantly.

Use the lightweight inline controls or the extension popup in the Chrome toolbar to start reading, pause, resume, or stop playback.

### Features
- Read highlighted text on almost any webpage
- Pause and resume playback
- Stop playback at any time
- Keep listening even after clicking away from the selected text
- Simple toolbar popup controls in Chrome
- Uses built-in browser speech synthesis for a lightweight experience

### Good for
- Reading articles out loud
- Listening to emails, docs, and web pages
- Accessibility and focus support
- Multitasking while browsing

### Privacy
Text Reader processes highlighted text locally in your browser and does not send user content to remote servers.
